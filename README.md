## Overview
Iron Lady is a **full-stack web application** built with **Flask, HTML, CSS, and JavaScript** to manage **customers, enrollments, programs, and payments** for training courses.  

The platform provides a **dynamic CRUD interface** to add, update, view, and delete customer records, supporting **multiple programs per customer**. It features **interactive tables, colorful UI, marquee announcements, dropdowns, date pickers, and progress tracking**, making it functional and visually appealing on laptops and desktops.

---

## Key Features
- Add, update, delete, and view customer information  
- Manage multiple programs per customer with **progress tracking**  
- Track **fees, payment status, and enrollment duration**  
- **Interactive tables** with hover effects  
- **Responsive design** for desktops and laptops  
- **Colorful, animated UI** with gradients, cards, and marquee announcements  

---

## Tech Stack
- **Backend:** Flask (Python)  
- **Frontend:** HTML, CSS, JavaScript  
- **Data Storage:** JSON (local file)  
- **UI/UX:** Responsive design, animations, hover effects, and cards
